# HAVEN — Sustainable Modular Architecture

A premium and minimalistic landing page concept for sustainable modular cabins.  
HAVEN is designed to showcase modern eco-friendly architecture with immersive interactions, elegant typography, and smooth animations.

---

## 🌿 Overview

HAVEN is a fully responsive landing page built using **HTML, CSS, and Vanilla JavaScript**.  
The project focuses on delivering a luxury minimalist experience inspired by nature, architecture, and mindful living.

The website includes:

- Smooth scrolling navigation
- Interactive hero section
- 3D parallax hover effects
- Mobile responsive menu
- Scroll reveal animations
- Premium glassmorphism UI
- Newsletter subscription interaction
